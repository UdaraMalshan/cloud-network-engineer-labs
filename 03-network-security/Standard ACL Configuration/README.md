# Standard ACL configuration (CCNA) - Cisco ASA firewall

Tool:
Cisco Packet Tracer

Concepts Practiced:
- Numbered and Named standard ACLs.
- Standard ACLs use near to the destination.
  

Commands Used:

- access-list 1 {PERMIT | DENY} {SRC_IP} {SRC SUBNET_MASK}
- ip access-group 1 {IN | OUT}
- show access-lists

  

Result:

 Successfully blocked packeet trasfering using source ip address.
